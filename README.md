# Malak Mohamed Mostafa Portfolio

Professional Angular portfolio converted from a CV into a responsive single-page portfolio.

## Features

- Angular standalone app structure
- Responsive hero, project gallery, skills, timeline, and contact sections
- Project filtering with Angular signals
- CV content organized as typed data in `src/app/app.ts`

## Run Locally

```bash
npm install
npm start
```

Open `http://localhost:4200/`.

## Build

```bash
npm run build
```
